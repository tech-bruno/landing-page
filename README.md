💡 Sobre o projeto

Esta landing page foi criada com o objetivo de simular uma página de vendas profissional para um curso de programação, contendo:

- Estrutura completa de marketing digital  
- Seções estratégicas de conversão  
- Design moderno e responsivo  
- Experiência do usuário (UX) otimizada  

---

🧱 Estrutura da página

- Hero (chamada principal)
- Benefícios
- Sobre o curso
- Módulos
- Depoimentos
- Oferta
- Garantia
- FAQ
- Footer

---

🛠️ Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript  

---

⚙️ Funcionalidades

- Layout 100% responsivo  
- Scroll suave  
- Animações ao rolar a página  
- Menu mobile (hamburger)  
- Botão flutuante de WhatsApp  
- Estrutura otimizada para conversão
